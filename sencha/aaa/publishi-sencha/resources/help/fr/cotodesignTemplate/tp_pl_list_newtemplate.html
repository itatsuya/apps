Déterminer la taille et la méthode d’impression d’un modèle

TYPE D’IMPRESSION
Sélectionnez le type de machine avec laquelle ce modèle sera imprimé.
Lors de l’impression sur une imprimante, sélectionnez « Impression couleur ». Lors de l’utilisation d’une machine de marquage laser sur films, sélectionnez « Impression sur film ». Lors de l’utilisation d’une imprimante métal, sélectionnez « Impression métallique ». Lors de l’utilisation d’une machine à découper, sélectionnez « Coupe ».
* « TYPE D’IMPRESSION » ne peut pas être modifié lors de l’édition d’un modèle enregistré.


 Lorsque « Impression sur film » est sélectionné pour « TYPE D’IMPRESSION »

Activer le contour
Imprime les contours des caractères, des images et des cliparts.
Lorsque vous cochez cette case, les contours deviennent plus précis, mais le temps d’impression s’allonge.

 Lorsque l’option « Impression métallique » est sélectionnée pour « TYPE D’IMPRESSION »

Matériau
Sélectionnez le matériau. « Matériau » est défini dans « Paramètres du matériau » dans cotodesign Tool.

Mode d’impression
Sélectionnez le mode correspondant à la cible d’impression.
Les conditions d’impression telles que la vitesse et la force d’impression varient en fonction du mode sélectionné.
Les conditions d’impression de chaque mode sont définies dans « Paramètres du matériau » dans cotodesign Tool 

Bidirection
Cochez cette case pour raccourcir le temps d’impression.

Activer le contour
Permet de faire une impression métallique des contours des caractères, des images et des cliparts.

 Lorsque « Coupe » est sélectionné pour « TYPE D’IMPRESSION »

Matériau
Sélectionnez le matériau et la couleur du matériau. « Matériau » est défini dans « Réglages des matériaux de coupe » dans cotodesign Tool.
