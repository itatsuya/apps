LIGNE DE COUPE

* Ceci n’est pas affiché pour les modèles destinés à l’impression sur film ou à l’impression métallique.

Ajoute une ligne de coupe au modèle.
Lorsque les données sont envoyées à une imprimante disposant d’une fonctionnalité permettant de découper le support, le support est alors découpé le long de la ligne de coupe. Les lignes de coupe sont ignorées par les imprimantes ne disposant pas de la fonctionnalité de découpe.

Autoriser pour cotodesign Client
Cochez cette case pour permettre aux clients d’ajouter des lignes de coupe.
Lorsque les modèles sélectionnés avec cette case sont ouverts dans cotodesign Client, la fonction de ligne de coupe est affichée et les clients peuvent ajouter de nouvelles lignes de coupe.

Lors de l’utilisation de la fonction de découpe, placez les données dans un gabarit qui peut être imprimé sur une imprimante équipée d’une fonction de découpe.

Onglet [TEXTE]

  • Entrée texte
    Saisir des caractères.
    Les lignes de contour des caractères saisis seront insérées dans l’écran de modification.
  • POLICE
    Sélectionne la police de caractère.
  • STYLE
    Sélectionne le style de caractère.
    Vous ne pouvez sélectionner que les styles applicables à la police. Par exemple, vous ne pouvez pas sélectionner le style italique pour une police qui ne dispose pas de version italique. De la même manière, vous ne pouvez pas sélectionner les styles désactivés. Vous pouvez activer/désactiver les styles dans cotodesign Tool.
      
  • ALIGNEMENT
    Aligne les positions horizontales des caractères saisis.
