TEXTE

* Ceci n’est pas affiché pour les modèles destinés à la découpe.

Entrée texte
Saisir des caractères.
Les caractères saisis seront insérés dans l’écran de modification.

TEXTE COURBE
Dispose les caractères le long d’une ligne courbe.
Cochez cette case pour afficher les éléments des paramètres.
Faites glisser le curseur « RAYON » pour spécifier le rayon du cercle. L’unité est en millimètres.
Vous pouvez également choisir d’aligner les caractères à l’extérieur ou à l’intérieur du cercle.

POLICE
Sélectionne la police de caractère.
(Les formats supportés sont TTF, WOFF et SVG. Enregistrez les polices WOFF avec les polices SVG.)

STYLE
Sélectionne le style de caractère.
Vous ne pouvez sélectionner que les styles applicables à la police. Par exemple, vous ne pouvez pas sélectionner le style italique pour une police qui ne dispose pas de version italique. De la même manière, vous ne pouvez pas sélectionner les styles désactivés. Vous pouvez activer/désactiver les styles dans cotodesign Tool.
  

ALIGNER
Sélectionne l’alignement.
