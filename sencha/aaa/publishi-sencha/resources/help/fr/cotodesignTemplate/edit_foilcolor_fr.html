COULEUR DE FILM

* Ceci s’affiche uniquement pour les modèles destinés à l’impression sur film.

Autoriser pour cotodesign Client
Cochez cette case si vous voulez que les clients sélectionnent la couleur de film dans cotodesign Client.
Décochez cette case si vous voulez que les clients sélectionnent un motif d’impression incluant la couleur de film.

Sélection de la couleur
Cliquez sur une couleur de film pour la spécifier. Utilisez cotodesign Tool pour définir les couleurs de film qui sont affichées.
Les images et les caractères s’affichent dans la couleur sélectionnée pour permettre de créer des modèles tout en vérifiant un aperçu semblable au produit fini.
