FOND

Spécifiez le motif ou le texte qui a déjà été appliqué au produit ou la couleur du produit lui-même pour reproduire à l’écran l’état du produit avant l’impression. Les objets insérés en fond ne s’affichent qu’à l’écran et ne seront pas imprimés.
Cette fonction permet de placer les images et le texte que vous souhaitez imprimer, tout en évitant les motifs et le texte qui existent déjà. De plus, il est possible de modifier la couleur du texte pour le différencier du produit.

Autoriser pour cotodesign Client
Permet l’ajout d’images de fond lorsque les clients personnalisent les données d’impression dans cotodesign Client.
Pour désactiver cette option, décochez la case.

Afficher la grille
Affiche une grille avec l’origine dans le coin inférieur gauche du modèle.
L’espacement entre les lignes étroites est de 5 mm. L’espacement entre les lignes épaisses est de 25 mm.
L’option est activée lorsque la case « Autoriser pour cotodesign Client » est cochée.

Onglet [IMAGE]
Affiche une liste des images qui peuvent être utilisées comme fond.
Cliquez sur une image pour l’insérer dans l’écran de modification.

Onglet [COULEUR]
Colorie l’ensemble du modèle avec la couleur spécifiée.

  • Pour effacer la couleur, cliquez sur  .
  • « RESTAURER LES VALEURS PAR DÉFAUT » : Revient aux paramètres « Couleur du matériau » définis dans cotodesign Tool.
    * Ceci n’est affiché que pour les modèles destinés à l’impression métallique.


Ajoute une image qui peut être placée dans le modèle.
L’écran pour sélectionner comment importer l’image s’affiche.

Sélectionner l’image depuis cet appareil
Ajoute une image enregistrée sur l’ordinateur utilisé.

Télécharger l’image depuis un appareil
Ajoute une image enregistrée sur une tablette, smartphone ou autre terminal d’information.

  1. Cliquez sur ce bouton pour afficher le premier code QR.
  2. Utilisez un terminal d’information pour lire le code QR affiché.
  3. Utilisez le SSID lu et le mot de passer pour connecter le terminal d’information au réseau.
    Si vous ne pouvez pas lire le code QR, utilisez le SSID et les chaînes de caractères du mot de passe affichées à l’écran.
  4. Cliquez sur [SUIVANT] pour afficher le deuxième code QR.
  5. Utilisez un terminal d’information pour lire le code QR affiché.
  6. Utilisez le terminal d’information pour accéder à l’URL.
    Si vous ne pouvez pas lire le code QR, accédez à l’URL affiché à l’écran.
  7. Téléchargez l’image.
