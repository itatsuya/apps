Ajoute une image qui peut être placée dans le modèle.
L’écran pour sélectionner comment importer l’image s’affiche.


Utiliser cet appareil
Ajoute une image enregistrée sur l’ordinateur utilisé.

Utiliser un autre appareil
Ajoute une image enregistrée sur une tablette, un smartphone ou un autre terminal d’information.

  1. Cliquez sur ce bouton pour afficher le premier code QR.
  2. Utilisez un terminal d’information pour lire le code QR affiché.
  3. Utilisez le SSID lu et le mot de passer pour connecter le terminal d’information au réseau.
    Si vous ne pouvez pas lire le code QR, utilisez le SSID et les chaînes de caractères du mot de passe affichées à l’écran.
  4. Cliquez sur [SUIVANT] pour afficher le deuxième code QR.
  5. Utilisez un terminal d’information pour lire le code QR affiché.
  6. Utilisez le terminal d’information pour accéder à l’URL.
    Si vous ne pouvez pas lire le code QR, accédez à l’URL affiché à l’écran.
  7. Téléchargez l’image.
* Pour les modèles destinés à l’impression sur film, les images sont affichées en deux couleurs (la couleur du film et le blanc).
Pour régler la partie colorée et le seuil de blanc, cochez la case « MODIFICATION D’IMAGE », puis cliquez sur l’image.

* Pour les modèles destinés à l’impression métallique, les images sont affichées en niveaux de gris.
Pour régler la luminosité ou extraire le contour, placez l’image, puis cliquez sur .
