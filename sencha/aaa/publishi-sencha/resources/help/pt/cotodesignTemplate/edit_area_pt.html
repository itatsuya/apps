ÁREA A EDITAR

Define no modelo uma moldura para introduzir imagens e carateres.
Ao especificar esta moldura em um modelo, apenas poderá introduzir imagens e carateres dentro da moldura. Não é mais possível introduzir imagens e carateres fora da moldura.
As imagens e carateres inseridos na área de edição podem ser editados com o cotodesign Client.

  Modelos dedicados a uso com impressão a cores
Existem três tipos de molduras que podem ser especificados. "ÁREA DA IMAGEM", "ÁREA DO TEXTO" e "ÁREA LIVRE".

ÁREA DA IMAGEM
Cria uma moldura para introduzir imagens e cliparts.
Clique (ou toque) neste botão para adicionar uma moldura preta. Para alterar o tamanho ou posição da moldura, arraste (ou percorra com o dedo) o item correspondente na interface.
É possível especificar várias molduras.

ÁREA DO TEXTO
Cria uma moldura para introduzir carateres.
Clique neste botão para adicionar uma moldura azul. Para alterar o tamanho ou posição da moldura, arraste (ou percorra com o dedo) o item correspondente na interface.
É possível especificar várias molduras.

ÁREA LIVRE
Cria uma moldura para introdução de objetos (imagens, cliparts e carateres).
É possível introduzir qualquer coisa nesta moldura desde que o item introduzido seja um objeto que possa ser adicionado ao cotodesign.
Clique neste botão para adicionar uma moldura verde. Para alterar o tamanho ou posição da moldura, arraste (ou percorra com o dedo) o item correspondente na interface.
É possível especificar várias molduras.

 Modelos dedicados a uso com impressão de películas

ÁREA DA PELÍCULA
Cria uma moldura para introdução de objetos (imagens, cliparts e carateres).
É possível introduzir qualquer coisa nesta moldura desde que o item introduzido seja um objeto que possa ser adicionado ao cotodesign.
Clique neste botão para adicionar uma moldura amarela. Para alterar o tamanho ou posição da moldura, arraste (ou percorra com o dedo) o item correspondente na interface.
É possível especificar várias molduras.
Ao criar um modelo dedicado para utilização com impressão em película, uma área de impressão (80 mm × 80 mm) e uma área de edição (50 mm × 50 mm) são criadas no centro do modelo.

Ao criar um modelo dedicado para utilização com impressão em metais, uma área de impressão em metais (80 mm × 80 mm) e uma área de edição (50 mm × 50 mm) são criadas no centro do modelo.
