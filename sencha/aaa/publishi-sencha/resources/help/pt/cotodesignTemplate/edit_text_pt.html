TEXTO

* Isto não é apresentado para modelos dedicados a uso com corte.

Entrada de texto
Introduza os carateres.
Os carateres introduzidos serão introduzidos no ecrã de edição.

TEXTO CURVO
Dispõe carateres ao longo de uma linha curva.
Marque esta caixa de seleção para apresentar os itens de definição.
Arraste o controlo deslizante "RAIO" para especificar o raio do círculo. A unidade é milímetros.
Também é possível selecionar se os carateres devem ser alinhados ao exterior ou ao interior do círculo.

FONTE
Seleciona a fonte do caráter.
(Os formatos suportados são TTF, WOFF e SVG. Registe os tipos de letra WOFF como um conjunto com os tipos de letra SVG.)

ESTILO
Seleciona o estilo do caráter.
Só é possível selecionar os estilos que sejam aplicáveis à fonte. Por exemplo, não é possível selecionar o estilo itálico com uma fonte que não tenha uma versão em itálico. Da mesma forma, não é possível selecionar estilos desativados. É possível ativar/desativar estilos na cotodesign Tool.
  

ALINHAR
Seleciona o alinhamento.
