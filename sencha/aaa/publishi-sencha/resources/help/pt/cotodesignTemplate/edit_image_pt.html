Adiciona uma imagem que pode ser posicionada no modelo.
O ecrã para selecionar como importar a imagem é apresentado.


Utilizar este dispositivo
Adiciona uma imagem guardada no computador atualmente em uso.

Utilizar outro dispositivo
Adiciona uma imagem guardada num tablet, smartphone ou terminal de informações.

  1. Clique neste botão para apresentar o primeiro código QR.
  2. Utilize um terminal de informações para ler o código QR apresentado.
  3. Utilize o SSID e a palavra-passe lidos para conectar o terminal de informações à rede.
    Se não quiser ler o código QR, utilize as cadeias de carateres SSID e palavra-passe apresentadas no ecrã.
  4. Clique em [SEGUINTE] para apresentar o segundo código QR.
  5. Utilize um terminal de informações para ler o código QR apresentado.
  6. Utilize o terminal de informações para aceder à URL lida.
    Se não quiser ler o código QR, aceda à URL apresentada no ecrã.
  7. Envie a imagem.
* Para modelos dedicados a uso com impressão de películas, as imagens são apresentadas a duas cores (a cor da película e branco).
Para ajustar a parte colorida e o limite de branco, selecione a caixa de verificação "EDIÇÃO DE IMAGEM" e clique na imagem.

* Para modelos dedicados a uso com impressão em metais, são exibidas imagens em tons de cinza.
Para ajustar o brilho ou extrair o contorno, coloque a imagem e clique em .
