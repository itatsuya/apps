L. CORTE

* Isto não é apresentado para modelos dedicados a uso com impressão de películas ou em metais.

Adiciona uma linha de corte ao modelo.
Quando os dados são enviados a uma impressora que tenha função de corte, o material é cortado ao longo da linha de corte. As linhas de corte são ignoradas por impressoras que não possuem funções de corte.

Ativar para o cotodesign Client
Marque esta caixa de verificação para permitir que os clientes adicionem linhas de corte.
Quando modelos com esta caixa de verificação são abertos no cotodesign Client, a função de linha de corte é apresentada, e os clientes podem adicionar novas linhas de corte.

Ao utilizar a função de corte, posicione os dados em um layout de impressão que possa gerar saída para uma impressora equipada com a função de corte.

Separador [TEXTO]

  • Entrada de texto
    Introduza os carateres.
    As linhas de contorno dos carateres introduzidos serão introduzidas no ecrã de edição.
  • FONTE
    Seleciona a fonte do caráter.
  • ESTILO
    Seleciona o estilo do caráter.
    Só é possível selecionar os estilos que sejam aplicáveis à fonte. Por exemplo, não é possível selecionar o estilo itálico com uma fonte que não tenha uma versão em itálico. Da mesma forma, não é possível selecionar estilos desativados. É possível ativar/desativar estilos na cotodesign Tool.
      
  • ALINHAMENTO
    Alinha as posições horizontais dos carateres introduzidos.
