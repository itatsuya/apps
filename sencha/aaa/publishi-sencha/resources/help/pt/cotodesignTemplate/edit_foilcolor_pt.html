Ativar para o cotodesign Client

Limpe esta caixa de verificação quando desejar que os clientes selecionem o design da impressão com a cor da película incluída.

Seleção de cores
Clique em uma cor de película para especificá-la. Use a cotodesign Tool para definir as cores da folha que são exibidas.
As imagens e carateres são apresentados na cor especificada para tornar possível criar modelos ao verificar uma pré-visualização próxima ao produto finalizado.
