FUNDO

Especifique o padrão ou texto que já foi aplicado ao produto ou a cor do próprio produto para reproduzir no ecrã o estado do produto antes da impressão. Os objetos inseridos como fundo só são apresentados no ecrã e não são impressos.
Esta função possibilita a colocação de imagens e texto que pretende imprimir enquanto evita os padrões e o texto já existentes e o destaque do texto ao alterar a cor para uma que difira do produto.

Ativar para o cotodesign Client
Permite a adição de imagens de fundo quando os clientes personalizam os dados de impressão no cotodesign Client.
Para não o permitir, desmarque a caixa de verificação.

Mostrar grade
Apresenta uma grelha com a origem no canto inferior esquerdo do modelo.
O espaçamento entre as linhas finas é 5 mm. O espaçamento entre as linhas grossas é 25 mm.
Está ativado quando a caixa de verificação "Ativar para o cotodesign Client" está selecionada.

Separador [IMAGEM]
Apresenta uma lista de imagens que podem ser utilizadas como fundo.
Clique em uma imagem para introduzi-la no ecrã de edição.

Separador [COR]
Pinta todo o modelo com a cor especificada.

  • Para limpar a cor, clique em  .
  • "RESTAURAR PADRÕES": Regressa à "Cor do Material" definida na cotodesign Tool.
    * Isto só é apresentado para modelos dedicados a uso com impressão em metais.


Adiciona uma imagem que pode ser posicionada no modelo.
O ecrã para selecionar como importar a imagem é apresentado.

Selecione imagem deste dispositivo
Adiciona uma imagem guardada no computador atualmente em uso.

Enviar imagem do dispositivo
Adiciona uma imagem guardada em um tablet, smartphone ou outro terminal de informações.

  1. Clique neste botão para apresentar o primeiro código QR.
  2. Utilize um terminal de informações para ler o código QR apresentado.
  3. Utilize o SSID e a palavra-passe lidos para conectar o terminal de informações à rede.
    Se não quiser ler o código QR, utilize as cadeias de carateres SSID e palavra-passe apresentadas no ecrã.
  4. Clique em [SEGUINTE] para apresentar o segundo código QR.
  5. Utilize um terminal de informações para ler o código QR apresentado.
  6. Utilize o terminal de informações para aceder à URL lida.
    Se não quiser ler o código QR, aceda à URL apresentada no ecrã.
  7. Envie a imagem.
