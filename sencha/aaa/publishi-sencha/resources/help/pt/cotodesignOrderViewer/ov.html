Estado do pedido

1. Pedidos não pagos
Esta é uma lista de pedidos que ainda não foram pagos.
Quando o pagamento for recebido, clique em  .
Quando o pagamento é recebido para um pedido, ele é movido para o campo "2. Aguardando a impressão", e pode ser impresso com o cotodesign Print Manager.

  1. Clique em um pedido para o qual o pagamento foi recebido.
  2. Veja as informações do pedido na parte inferior do ecrã e confirme que é a encomenda do cliente.
  3. Clique em  .
O status é ativado quando a caixa de verificação "Ignorar Confirmação de Pagamento" é limpa no separador [Pasta Pedido] do ecrã [Configurações] no cotodesign Print Manager.

2. Aguardando a impressão
Esta é a lista de ordens que estão a aguardar impressão.
Ao utilizar o cotodesign Print Manager para prosseguir com o processamento de impressões, os pedidos são movidos para o campo "3. Enviado para impressão".

3. Enviado para impressão
Esta é a lista de pedidos que estão a ser impressos ou que foram impressos.
Quando o produto tiver sido entregue ao cliente, clique em  .
Quando os pedidos são definidos como entregues, eles são movidos para o campo "4. Entregue".

  1. Clique em um pedido para o qual o produto foi entregue.
  2. Veja as informações do pedido na parte inferior do ecrã e confirme que é a encomenda do cliente.
  3. Clique em  .

4. Entregue
Esta é a lista de pedidos que foram entregues aos clientes, e que portanto concluíram o processamento.
