SCHNITTLINIE

* Dies wird bei Templates für den Folien- oder Metallaufdruck nicht angezeigt.

Fügt eine Schnittlinie zum Template hinzu.
Wenn Daten an einen Drucker gesendet werden, der eine Schnittfunktion für Medien besitzt, wird das Medium entlang der Schnittlinie zugeschnitten. Schnittlinien werden von Druckern, die keine Schnittfunktion besitzen, ignoriert.

Aktivieren für cotodesign Client
Aktivieren Sie dieses Kontrollkästchen, wenn Sie möchten, dass Kunden selbstständig Schnittlinien hinzufügen können.
Wenn Templates mit diesem Kontrollkästchen in cotodesign Client geöffnet werden, wird die Schnittlinienfunktion angezeigt und Kunden können neue Schnittlinien hinzufügen.

Platzieren Sie bei Verwendung der Schnittfunktion die Daten in einem Print-Layout, das an einen Drucker mit Schnittfunktion ausgegeben werden kann.

Registerkarte [TEXT]

  • Texteingabe
    Geben Sie Zeichen ein.
    Die Konturen der eingegebenen Zeichen werden am Bearbeitungsbildschirm eingefügt.
  • SCHRIFTART
    Wählt die Schriftart für die Zeichen aus.
  • STIL
    Wählt den Stil für die Zeichen aus.
    Sie können nur die Stile auswählen, die für die Schriftart verwendet werden können. So können Sie beispielsweise den Kursiv-Stil nicht für eine Schriftart auswählen, die keine kursive Variante hat. Zudem können Sie auch keine deaktivierten Stile auswählen. Sie können Stile in cotodesign Tool aktivieren/deaktivieren.
      
  • AUSRICHTUNG
    Passt die horizontale Position der eingegebenen Zeichen an.

Registerkarte [LINIE]

  • RECHTECK/DREIECK/KREIS/FÜNFECK/SECHSECK/ABGERUNDET. RECHTECK/STERN/HERZ/TRAPEZ/RAUTE/SPRECHBLASE
    Klicken Sie hier, um eine Figur in den Bearbeitungsbildschirm einzufügen.
