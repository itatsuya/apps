BEARB.-BER.

Definieren Sie im Template einen Rahmen zum Einfügen von Bildern und Zeichen.
Wenn Sie diesen Rahmen in einem Template definieren, können Sie Bilder und Zeichen nur innerhalb dieses Rahmens einfügen. Sie können keine Bilder und Zeichen mehr außerhalb des Rahmens einfügen.
Die im Bearbeitungsbereich eingefügten Bilder und Zeichen können mit dem cotodesign Client bearbeitet werden.

  Templates für den Farbdruck
Es gibt drei Arten von Rahmen, die Sie definieren können. „BILDBEREICH“, „TEXTBEREICH“ und „FREIRAUM“.

BILDBEREICH
Erstellt einen Rahmen zum Einfügen von Bildern und Cliparts.
Klicken (oder tippen) Sie auf diese Schaltfläche, um einen schwarzen Rahmen hinzuzufügen. Zum Ändern der Größe oder Position des Rahmens ziehen (oder wischen) Sie die Bearbeitungspunkte in beliebige Richtungen.
Sie können mehrere Rahmen definieren.

TEXTBEREICH
Erstellt einen Rahmen zum Einfügen von Zeichen.
Klicken Sie auf diese Schaltfläche, um einen blauen Rahmen hinzuzufügen. Zum Ändern der Größe oder Position des Rahmens ziehen (oder wischen) Sie die Bearbeitungspunkte in beliebige Richtungen.
Sie können mehrere Rahmen definieren.

FREIRAUM
Erstellt einen Rahmen zum Einfügen beliebiger Objekte (Bilder, Cliparts und Zeichen).
Sie können alles in diesen Rahmen einfügen, vorausgesetzt das Element ist ein Objekt, das zu cotodesign hinzugefügt werden kann.
Klicken Sie auf diese Schaltfläche, um einen grünen Rahmen hinzuzufügen. Zum Ändern der Größe oder Position des Rahmens ziehen (oder wischen) Sie die Bearbeitungspunkte in beliebige Richtungen.
Sie können mehrere Rahmen definieren.

 Templates für den Foliendruck

FOLIENBEREICH
Erstellt einen Rahmen zum Einfügen beliebiger Objekte (Bilder, Cliparts und Zeichen).
Sie können alles in diesen Rahmen einfügen, vorausgesetzt das Element ist ein Objekt, das zu cotodesign hinzugefügt werden kann.
Klicken Sie auf diese Schaltfläche, um einen gelben Rahmen hinzuzufügen. Zum Ändern der Größe oder Position des Rahmens ziehen (oder wischen) Sie die Bearbeitungspunkte in beliebige Richtungen.
Sie können mehrere Rahmen definieren.
Wenn Sie ein Template für den Foliendruck erstellen, werden in der Mitte des Templates ein Druckbereich (80 mm × 80 mm) und ein Bearbeitungsbereich (50 mm × 50 mm) erstellt.

 Templates für den Metallaufdruck

METALLBEREICH
Erstellt einen Rahmen zum Einfügen beliebiger Objekte (Bilder, Cliparts und Zeichen).
Sie können alles in diesen Rahmen einfügen, vorausgesetzt das Element ist ein Objekt, das zu cotodesign hinzugefügt werden kann.
Klicken Sie auf diese Schaltfläche, um einen grauen Rahmen hinzuzufügen. Zum Ändern der Größe oder Position des Rahmens ziehen (oder wischen) Sie die Bearbeitungspunkte in beliebige Richtungen.

Sie können mehrere Rahmen definieren.
Wenn Sie ein Template für den Metallaufdruck erstellen, werden in der Mitte des Templates ein Metallaufdruckbereich (80 mm × 80 mm) und ein Bearbeitungsbereich (50 mm × 50 mm) erstellt.

 Templates für das Schneiden

SCHNEIDEBEREICH
Erstellt einen Rahmen zum Einfügen von Cliparts.
Klicken Sie auf diese Schaltfläche, um einen roten rechteckigen Rahmen hinzuzufügen. Zum Ändern der Größe oder Position des Rahmens ziehen (oder wischen) Sie die Bearbeitungspunkte in beliebige Richtungen.
Sie können mehrere Rahmen definieren.
