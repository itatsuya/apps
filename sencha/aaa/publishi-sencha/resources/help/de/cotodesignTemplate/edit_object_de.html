Bild- und Zeichenbearbeitungen

Auswählen
Klicken (oder tippen) Sie auf ein Bild oder ein Zeichen.
Wenn Sie ein Objekt auswählen, erscheinen Bearbeitungspunkte rundherum.


Verschieben, umformen, drehen

 Templates für den Farbdruck/Templates für das Schneiden
 

 Templates für den Metallaufdruck/Templates für den Foliendruck
 

Ziehen (oder wischen) Sie die Bearbeitungspunkte.

: Verschiebt das Objekt.
 Die Position wird automatisch so eingestellt, dass sie mit dem oberen, unteren, linken oder rechten Rand des anderen zu bewegenden Objekts übereinstimmt.
 Objekte können nur mit dem gleichen Rand ausgerichtet werden (Beispiel: oben und oben).
 Wenn Sie Objekte frei platzieren möchten, deaktivieren Sie das Kontrollkästchen "RICHTLINIE".
: Ändert die Form des Objekts.
: Dreht das Objekt.
: Erhöht/verringert die Größe des Objekts, wobei das Seitenverhältnis beibehalten wird.
* Eine übermäßige Vergrößerung kann das Bild rau machen und somit verhindern, dass die gewünschten Druckergebnisse erzielt werden. Seien Sie vorsichtig.


Klicken (oder tippen) Sie auf die Bearbeitungspunkte.
: Aktivieren Sie diese Option, wenn Sie die Position oder Größe des Objekts nicht verändern möchten.
: Verwenden Sie diese Option, um die Teile des Bildes auszublenden, die Sie nicht drucken möchten (es wird nur für Bilder angezeigt).
Wenn Sie diese Option aktivieren, werden um das Bild herum Bearbeitungspunkte zum Zuschneiden angezeigt.
: Nehmen Sie Anpassungen an Elementen wie der Bildhelligkeit vor und konfigurieren Sie Einstellungen, die sich auf die Kontur beziehen. (Dies wird für Templates für den Metallaufdruck und für Templates für den Foliendruck angezeigt.)
