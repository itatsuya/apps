Bestimmung der Größe und des Druckverfahrens für ein Template

DRUCKART
Wählen Sie den Gerätetyp aus, mit dem dieses Template gedruckt/auf das dieses Template gedruckt werden soll.
Wenn Sie einen Drucker verwenden, wählen Sie „Farbdruck“. Wählen Sie bei Verwendung eines Laser Decorators „Foliendruck“ aus. Wenn Sie einen Metalldrucker verwenden, wählen Sie „Metallaufdruck“. Wenn Sie eine Schneidemaschine verwenden, wählen Sie „Schneiden“.
* Die „DRUCKART“ kann nicht geändert werden, wenn Sie ein gespeichertes Template bearbeiten.


 Wenn „Foliendruck“ als „DRUCKART“ ausgewählt ist

Umriss aktivieren
Druckt die Umrisse von Zeichen, Bildern und Cliparts.
Wenn Sie dieses Kontrollkästchen aktivieren, werden die Umrisse klarer, aber die Druckzeit wird länger.

 Wenn „Metallaufdruck“ als „DRUCKART“ ausgewählt ist

Material
Wählen Sie das Material aus. „Material“ wird im cotodesign Tool unter „Materialeinstellungen“ eingestellt.

Druckmodus
Wählen Sie den Modus, der dem Ziel des Aufdrucks entspricht.
Die Druckbedingungen wie Geschwindigkeit und Aufdruckkraft ändern sich je nach ausgewähltem Modus.
Die Aufdruckbedingungen für jeden Modus werden mit „Materialeinstellungen“ im cotodesign Tool eingestellt. 

Bidirektionalität
Aktivieren Sie dieses Kontrollkästchen, um die Druckzeit zu verkürzen.

Umriss aktivieren
Erstellt einen Metallaufdruck aus den Umrissen von Zeichen, Bildern und Cliparts.

 Wenn „Schneiden“ als „DRUCKART“ ausgewählt ist

Material
Wählen Sie das Material und die Materialfarbe aus. „Material“ wird in cotodesign Tool unter „Einstellungen Schneidmaterial“ eingestellt.
