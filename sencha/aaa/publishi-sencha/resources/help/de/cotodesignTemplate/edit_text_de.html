* Dies wird bei Templates für das Schneiden nicht angezeigt.

Texteingabe
Geben Sie Zeichen ein.
Die eingegebenen Zeichen werden am Bearbeitungsbildschirm eingefügt.

GEKRÜMMTER TEXT
Ordnet die Figuren entlang einer geschwungenen Linie an.
Aktivieren Sie dieses Kontrollkästchen, um die Einstelloptionen anzuzeigen.
Ziehen Sie den Schieber „RADIUS“, um den Radius des Kreises festzulegen. Die Einheit ist Millimeter.
Sie können auch wählen, ob die Zeichen an der Außenseite oder der Innenseite des Kreises ausgerichtet werden sollen.

SCHRIFTART
Wählt die Schriftart für die Zeichen aus.
(Die unterstützten Formate sind TTF, WOFF und SVG. Registrieren Sie WOFF-Schriften als Set mit SVG-Schriften.)

STIL
Wählt den Stil für die Zeichen aus.
Sie können nur die Stile auswählen, die für die Schriftart verwendet werden können. So können Sie beispielsweise den Kursiv-Stil nicht für eine Schriftart auswählen, die keine kursive Variante hat. Zudem können Sie auch keine deaktivierten Stile auswählen. Sie können Stile in cotodesign Tool aktivieren/deaktivieren.
  

AUSRICHTEN
Wählt die Ausrichtung aus.
