訂單的狀態

1. 未付款訂單
未付款訂單的清單。
在接受付款之後，請按一下 。
完成付款確認之後，移至「2. 等待列印」欄位，即可在 cotodesign Print Manager 中進行列印。

  1. 按一下已接受付款的訂單。
  2. 檢視畫面下方的訂購資訊，確認為客戶的訂購。
  3. 按一下 。
在此狀態下，在 cotodesign Print Manager「設定」畫面的［購物車］標籤上，只有在「跳過付款確認」為關時才能生效。

2. 等待列印
等待列印的訂購清單。
使用 cotodesign Print Manager 進行列印處理時，將會移至「3. 列印中」的欄位。

3. 列印中
列印中或列印完成的訂購清單。
將產品交付給客戶之後，請按一下 。
完成交付之後，將會移至「4. 交件」的欄位。

  1. 按一下已將產品交貨的訂購。
  2. 檢視畫面下方的訂購資訊，確認為客戶的訂購。
  3. 按一下 。

4. 交件
已將產品交付給客戶，完成所有處理作業的訂購清單。
