SPOT COLOR

* This is displayed for templates dedicated for use with color printing.

PRIMER
Performs the undercoat processing.

WHITE
Turn this function on when you want to perform white printing.

GLOSS
Performs glossy printing.
You can select glossy or matte printing.
When you turn this function on, you can configure the special effects settings.

REVERSE PRINT
Inverts the printing order and prints a mirror image.
Use this when printing objects from the back and displaying objects from the front.
