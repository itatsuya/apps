[SPECIAL EFFECTS] tab
Sets the glossy printing of the selected object.
To apply glossy printing to all objects at once instead of individually, select the "APPLY TO ALL" check box.
When "MATTE Varnish" is selected for the glossy printing type, you can set the texture type.



Templates dedicated for use with metal imprinting

FILL SHARPNESS
Adjusts the number of dots to imprint on the metal (the dot density).
If the "FILL SHARPNESS" check box is cleared, the surface is set so that it is not filled.

DRAW OUTLINE
Makes a metal imprint of the outlines of characters, images, and clipart.
