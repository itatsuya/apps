Determining the Size and Printing Method for a Template

TYPE OF PRINTING
Select the type of machine that this template will be printed/imprinted with.
When printing on a printer, select "Color Print." When using a laser decorator, select "Foil Print." When using a metal printer, select "Metal Imprint." When using a cutting machine, select "Cutting."
* "TYPE OF PRINTING" cannot be changed when editing a template that has been saved.


 When "Foil Print" is selected for "TYPE OF PRINTING"

Enable Outline
Prints the outlines of characters, images, and clipart.
When you select this check box, outlines become clearer, but the printing time becomes longer.

 When "Metal Imprint" is selected for "TYPE OF PRINTING"

Material
Select the material. "Material" is set with "Material Settings" in cotodesign Tool.

Imprint Mode
Select the mode to match the imprinting target.
Imprinting conditions such as the speed and imprinting force will change depending on the selected mode.
The imprinting conditions of each mode are set with "Material Settings" in cotodesign Tool. 

Bidirection
Select this check box to shorten the imprinting time.

Enable Outline
Makes a metal imprint of the outlines of characters, images, and clipart.

 When "Cutting" is selected for "TYPE OF PRINTING"

Material
Select the material and the material color. "Material" is set with "Cutting Material Settings" in cotodesign Tool.
