BACKGROUND

Specify the pattern or text that has already been applied to the product or the color of the product itself to reproduce on the screen the state of the product prior to printing. Objects inserted as the background are only displayed on the screen and are not printed.
This function makes it possible to place the pictures and text that you want to print while avoiding the patterns and text that already exist and to make text stand out by changing its color to one that differs from the product.

Enable for cotodesign Client
Permits the addition of background images when customers customize the printing data in cotodesign Client.
To not permit this, clear the check box.

Show grid
Displays a grid with the origin at the lower-left corner of the template.
The spacing between the narrow lines is 5 mm (0.2 in.). The spacing between the thick lines is 25 mm (1.0 in.).
This is enabled when the "Enable for cotodesign Client" check box is selected.

[IMAGE] tab
Displays a list of the images that can be used as the background.
Click an image to insert it in the editing screen.

[COLOR] tab
Colors the whole template with the specified color.

  • To clear the color, click  .
  • "RESTORE DEFAULTS": Returns to the "Material Color" set in cotodesign Tool.
    * This is only displayed for templates dedicated for use with metal imprinting.


Adds an image that can be placed in the template.
The screen for selecting how to import the image is displayed.

Select image from this device
Adds an image saved on the computer being used.

Upload image from device
Adds an image saved on a tablet, smartphone, or other information terminal.

  1. Click this button to display the first QR code.
  2. Use an information terminal to read the displayed QR code.
  3. Use the read SSID and password to connect the information terminal to the network.
    If you cannot read the QR code, use the SSID and password character strings displayed in the screen.
  4. Click [NEXT] to display the second QR code.
  5. Use an information terminal to read the displayed QR code.
  6. Use the information terminal to access the read URL.
    If you cannot read the QR code, access the URL displayed in the screen.
  7. Upload the image.
