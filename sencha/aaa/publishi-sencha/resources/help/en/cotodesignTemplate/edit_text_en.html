TEXT

* This is not displayed for templates dedicated for use with cutting.

Text input
Enter characters.
The entered characters will be inserted in the editing screen.

CURVED TEXT
Lays out characters along a curved line.
Select this check box to display the setting items.
Drag the "RADIUS" slider to specify the radius of the circle. The unit is millimeters.
You can also select whether to align the characters with the outside or the inside of the circle.

FONT
Selects the character font.
(The supported formats are TTF, WOFF, and SVG. Register WOFF fonts as a set with SVG fonts.)

STYLE
Selects the character style.
You can only select the styles that are applicable for the font. For example, you cannot select the italic style with a font that does not have an italic version. In the same manner, you cannot select disabled styles. You can enable/disable styles in cotodesign Tool.
  

ALIGN
Selects the alignment.
