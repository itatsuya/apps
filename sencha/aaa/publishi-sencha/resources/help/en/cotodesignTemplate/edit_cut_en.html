CUT LINE

* This is not displayed for templates dedicated for use with foil printing or metal imprinting.

Adds a cutting line to the template.
When the data is sent to a printer that has a function for cutting the media, the media is cut along the cutting line. Cutting lines are ignored by printers that do not have cutting functions.

Enable for cotodesign Client
Select this check box to enable customers to add cutting lines.
When templates with this check box selected are opened in cotodesign Client, the cutting line function is displayed and customers can add new cutting lines.

When using the cutting function, place the data in a print layout that can be output to a printer equipped with a cutting function.

[TEXT] Tab

  • Text input
    Enter characters.
    The contour lines of the entered characters will be inserted in the editing screen.
  • FONT
    Selects the character font.
  • STYLE
    Selects the character style.
    You can only select the styles that are applicable for the font. For example, you cannot select the italic style with a font that does not have an italic version. In the same manner, you cannot select disabled styles. You can enable/disable styles in cotodesign Tool.
      
  • ALIGNMENT
    Aligns the horizontal positions of the entered characters.
