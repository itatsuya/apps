FOIL COLOR

* This is only displayed for templates dedicated for use with foil printing.

Enable for cotodesign Client
Select this check box when you want customers to select the foil color from cotodesign Client.
Clear this check box when you want customers to select the printing design with the foil color included.

Color selection
Click a foil color to specify it. Use cotodesign Tool to set the foil colors that are displayed.
Images and characters are displayed in the specified color to make it possible to create templates while checking a preview close to the finished product.
