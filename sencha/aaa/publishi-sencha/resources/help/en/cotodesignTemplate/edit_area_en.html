EDIT AREA

Set in the template a frame for inserting images and characters.
When you specify this frame in a template, you can only insert images and characters within the frame. You can no longer insert images and characters outside of the frame.
The images and characters inserted in the edit area can be edited with cotodesign Client.

  Templates dedicated for use with color printing
There are three types of frames that you can specify. "IMAGE AREA," "TEXT AREA," and "FREE AREA."

IMAGE AREA
Creates a frame for inserting images and clipart.
Click (or tap) this button to add one black frame. To change the size or position of the frame, drag (or swipe) its interface item.
You can specify multiple frames.

TEXT AREA
Creates a frame for inserting characters.
Click this button to add one blue frame. To change the size or position of the frame, drag (or swipe) its interface item.
You can specify multiple frames.

FREE AREA
Creates a frame for inserting objects (images, clipart, and characters).
You can insert anything into this frame provided the inserted item is an object that can be added to cotodesign.
Click this button to add one green frame. To change the size or position of the frame, drag (or swipe) its interface item.
You can specify multiple frames.

 Templates dedicated for use with foil printing

FOIL AREA
Creates a frame for inserting objects (images, clipart, and characters).
You can insert anything into this frame provided the inserted item is an object that can be added to cotodesign.
Click this button to add one yellow frame. To change the size or position of the frame, drag (or swipe) its interface item.
You can specify multiple frames.
When you create a template dedicated for use with foil printing, one printing area (80 mm × 80 mm; 3.2 in. × 3.2 in.) and one edit area (50 mm × 50 mm; 2.0 in. × 2.0 in.) are created in the center of the template.

 Templates dedicated for use with metal imprinting

METAL AREA
Creates a frame for inserting objects (images, clipart, and characters).
You can insert anything into this frame provided the inserted item is an object that can be added to cotodesign.
Click this button to add one gray frame. To change the size or position of the frame, drag (or swipe) its interface item.

You can specify multiple frames.
When you create a template dedicated for use with metal imprinting, one metal imprinting area (80 mm × 80 mm; 3.2 in. × 3.2 in.) and one edit area (50 mm × 50 mm; 2.0 in. × 2.0 in.) are created in the center of the template.

 Templates dedicated for use with cutting

CUT AREA
Creates a frame for inserting clipart.
Click this button to add one red rectangular frame. To change the size or position of the frame, drag (or swipe) its interface item.
You can specify multiple frames.
