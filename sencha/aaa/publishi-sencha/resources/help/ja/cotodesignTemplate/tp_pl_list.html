Template一覧 と Print Layout一覧

cotodesign に登録されている「Template」と「Print Layout」の一覧です。

Templateとは
お客様に選んでいただく、印刷データのひな形のことです。
商品サイズ、印刷する文字や画像、特殊加工の有無を設定できます。

Print Layoutとは
プリンターで印刷するときの、Templateの配置を設定したデータです。
Print Layoutに配置されていないTemplateは印刷できません。
通常、Print Layoutには複数のTemplateが配置できるように設定します。いくつものTemplateを配置した印刷データが1つ作られ、プリンターで印刷されます。

名前
登録されているTemplateのファイル名を表示します。
Templateのファイル名には命名規則があり、ピリオドで区切って階層構造を示しています。

印刷種類
プリンターで印刷する場合は「カラー印刷」、レーザー箔転写機の場合は「箔印刷」、メタルプリンターの場合は「打刻」、カッティングマシンの場合は「カット」を選択します。
「▼」をクリックし、「印刷種類」を選択することで、Template一覧に表示するTemplateファイルを絞り込みます。

特殊効果
「特色」を使用してTemplateファイルを作成した場合に特殊効果に応じたアイコンが表示されます。

Print Layout一覧


新しい Print Layout を作ります。


Print Layout の設定を変えます。


SVGファイルをインポートし、新しい Print Layout を追加します。
SVGファイルの仕様については、『cotodesign セットアップガイド』を参照してください。


Print Layoutのプレビューを表示します。

Print Layout を削除します。


Print Layoutファイルをインポートし、サーバーへアップロードします。


選択したPrint Layoutを保存します。

配置先
Templateを配置するPrint Layoutを選びます。
Templateには、配置先のPrint Layoutが少なくとも1つ必要です。配置先の無いTemplateは、印刷することができません。
1つのTemplateに対して配置先のPrint Layoutを複数選ぶことができますが、カラー印刷専用Templateは、カラー印刷のPrint Layoutだけに関連付けできるように、TemplateとPrint Layoutは、同じ印刷種類同士でだけ、組み合わせることができます。注文されたTemplateをどのPrint Layoutに配置するかは、cotodesign Print Managerで設定します。

標準
cotodesign Print Manager の［標準のPrint Layoutへ自動配置］がオンのとき、自動配置される Print Layout を選びます。

名前
Print Layout のファイル名を表示します。

印刷種類
プリンターで印刷する場合は「カラー印刷」、レーザー箔転写機の場合は「箔印刷」、メタルプリンターの場合は「打刻」、カッティングマシンの場合は「カット」を選択します。

「▼」をクリックし、「印刷種類」を選択することで、Print Layout一覧に表示するファイルを絞り込みます。

保存
Template と Print Layout の配置先の設定をサーバーに保存します。
