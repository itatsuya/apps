COLOR DE PUNTO

* Esto se muestra en el caso de las plantillas específicas para la impresión en color.

IMPRIMACIÓN
Realiza el procesamiento dar una primera capa.

BLANCO
Activar esta función para realizar una impresión en blanco.

BRILLANTE
Realiza una impresión con brillo.
Puede seleccionar una impresión brillante o mate.
Al activar esta función, puede configurar los ajustes de efectos especiales.

IMPRESIÓN INVERSA
Invierte el orden de impresión e imprime una imagen reflejada.
Usarlo cuando se imprimen objetos reflejados y se muestran con la vista frontal.
