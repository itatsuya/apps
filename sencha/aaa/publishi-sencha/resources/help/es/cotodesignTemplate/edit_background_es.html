FONDO

Especifique el patrón o texto que ya se ha aplicado al producto o el color del propio producto para reproducir en la pantalla el estado de este antes de la impresión. Los objetos insertados como fondo solo se visualizan en la pantalla y no se imprimen.
Esta función permite colocar las imágenes y el texto que se desea imprimir evitando los patrones y el texto ya existentes y hacer que el texto destaque cambiando su color por uno diferente al del producto.

Habilitar para cotodesign Client
Permite la adición de imágenes de fondo cuando los clientes personalizan los datos de impresión en el cotodesign Client.
Para no permitir esto, desactive la casilla de verificación.

Mostrar cuadrícula
Muestra una cuadrícula con el origen en la esquina inferior izquierda de la plantilla.
El espaciado entre las líneas estrechas es de 5 mm. El espaciado entre las líneas gruesas es de 25 mm.
Esta opción se activa cuando se selecciona la casilla de verificación «Habilitar para cotodesign Client».

Pestaña [IMAGEN]
Muestra una lista de las imágenes que se pueden utilizar como fondo.
Pinchar una imagen para introducirla en la pantalla de edición.

Pestaña [COLOR]
Colorea toda la plantilla con el color especificado.

  • Para eliminar el color, pinchar en  .
  • «RESTAURAR VALORES PREDETERMINADOS»: Vuelve a la opción «Color de material» de la herramienta cotodesign Tool
    * Esto sólo se muestra para las plantillas específicas para la impresión en metal.


Añade una imagen que se puede colocar en la plantilla.
Se muestra la plantilla para seleccionar cómo importar la imagen.

Seleccionar imagen de este dispositivo
Añade una imagen guardada en el ordenador que se está utilizando.

Cargar imagen desde el dispositivo
Añade una imagen guardada en una tableta, un teléfono inteligente u otro terminal de información.

  1. Pinche en este botón para mostrar el primer código QR.
  2. Use un terminal de información para leer el código QR mostrado.
  3. Usar el SSID de lectura y la contraseña para conectar el terminal de información a la red.
    Si no puede leer el código QR, utilice el SSID y las cadenas de caracteres de contraseña que aparecen en la pantalla.
  4. Pinche en [SIGUIENTE] para mostrar el segundo código QR.
  5. Use un terminal de información para leer el código QR mostrado.
  6. Usar el terminal de información para acceder a la URL de lectura.
    Si no puede leer el código QR, acceda a la URL que se muestra en la pantalla.
  7. Cargar la imagen.
