TEXTO

* Esto no se muestra en el caso de las plantillas específicas para el corte.

Entrada de texto
Introduce caracteres.
Los caracteres se introducen en la pantalla de edición.

TEXTO CURVADO
Distribuye los caracteres a lo largo de una línea curva.
Marque esta casilla de verificación para mostrar los elementos de configuración.
Arrastre el control deslizante «RADIO» para especificar el radio del círculo. La unidad son milímetros.
También puede seleccionar si desea alinear los caracteres con el exterior o el interior del círculo.

FUENTE
Selecciona la fuente de caracteres.
(Los formatos admitidos son TTF, WOFF y SVG. Registrar fuentes WOFF como un conjunto con fuentes SVG.)

ESTILO
Selecciona el estilo de los caracteres.
Solo puede seleccionar los estilos que sean aplicables a la fuente. Por ejemplo, no se puede seleccionar la cursiva con una fuente que no tenga una versión cursiva. De la misma manera, no puede seleccionar estilos desactivados. Puede activar/desactivar estilos en cotodesign Tool.
  

ALINEAR
Selecciona la alineación
