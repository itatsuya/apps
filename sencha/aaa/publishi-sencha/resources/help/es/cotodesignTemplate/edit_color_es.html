Plantillas específicas para la impresión en color

 Cuando la casilla de verificación «BRILLANTE» bajo «COLOR DE PUNTO» está desactivada

Llenar/línea
Selecciona si cambia el color de relleno o el color de la línea.
 

Selección de color
Pinchar un color para seleccionarlo. Utilice cotodesign Tool para configurar los colores que se muestran en la paleta de colores.
Para borrar el color, haga clic en  .
 se muestra cuando la casilla de verificación [Añadir Ninguno a la paleta] esté seleccionada en [Paleta de colores] en cotodesign Tool.

NO IMPRIMIR COLOR
Seleccione esta casilla de verificación para utilizar el color natural del objeto en el que se va a imprimir y no aplicar ninguna tinta.



 Cuando la casilla de verificación «BRILLANTE» bajo «COLOR DE PUNTO» está seleccionada

Pestaña [COLOR]

    Llenar/línea
    Selecciona si cambia el color de relleno o el color de la línea.
     

    Selección de color
    Pinchar un color para seleccionarlo. Utilice cotodesign Tool para configurar los colores que se muestran en la paleta de colores.
    Para borrar el color, haga clic en  .
     se muestra cuando la casilla de verificación [Añadir Ninguno a la paleta] esté seleccionada en [Paleta de colores] en cotodesign Tool.

    NO IMPRIMIR COLOR
    Seleccione esta casilla de verificación para utilizar el color natural del objeto en el que se va a imprimir y no aplicar ninguna tinta.

Pestaña [EFECTOS ESPECIALES]
Define la impresión brillante del objeto seleccionado.
Para aplicar la impresión brillante a todos los objetos a la vez, en lugar de individualmente, seleccione la casilla de verificación «APLICA A TODO».
Cuando se selecciona «BARNIZ MATE» para el tipo de impresión brillante, se puede ajustar el tipo de textura.



Plantillas específicas para la impresión en metal

PRECISIÓN DE LLENADO
Ajusta el número de puntos para imprimir en el metal (la densidad de puntos).
Si la casilla de verificación «PRECISIÓN DE LLENADO» está desactivada, la superficie se configura de forma que no se rellene.

DIBUJAR CONTORNO
Realiza una impresión metálica de los contornos de caracteres, imágenes y imágenes prediseñadas.
