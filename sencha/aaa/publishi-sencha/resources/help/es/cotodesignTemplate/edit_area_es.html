ÁREA A EDITAR

Crea un marco en la plantilla para introducir imágenes y caracteres.
Si introduce este marco en una plantilla, las imágenes y caracteres solo se pueden colocar dentro de dicho marco. No se admite ponerlos fuera de él.
Las imágenes y caracteres insertados en el área de edición se pueden editar con cotodesign Client.

  Plantillas específicas para la impresión en color
Hay tres tipos de marcos que pueden especificar. «ÁREA DE IMAGEN», «ÁREA DE TEXTO» y «ÁREA LIBRE»

ÁREA DE IMAGEN
Crea un marco para insertar imágenes y imágenes prediseñadas.
Haga clic (o toque) en este botón para añadir un marco negro. Para modificar el tamaño o la posición del marco, arrastre (o deslice) este elemento de interfaz.
Puede indicar múltiples marcos.

ÁREA DE TEXTO
Crea un marco para insertar caracteres.
Haga clic en este botón para añadir un marco azul. Para modificar el tamaño o la posición del marco, arrastre (o deslice) este elemento de interfaz.
Puede indicar múltiples marcos.

ÁREA LIBRE
Crea un marco para insertar objetos (imágenes, imágenes prediseñadas y caracteres).
En este marco puede insertar cualquier cosa siempre y cuando el elemento introducido sea un objeto que se pueda añadir a cotodesign.
Haga clic en este botón para añadir un marco verde. Para modificar el tamaño o la posición del marco, arrastre (o deslice) este elemento de interfaz.
Puede indicar múltiples marcos.

 Plantillas específicas para la impresión en lámina

ÁREA DE LÁMINA
Crea un marco para insertar objetos (imágenes, imágenes prediseñadas y caracteres).
En este marco puede insertar cualquier cosa siempre y cuando el elemento introducido sea un objeto que se pueda añadir a cotodesign.
Haga clic en este botón para añadir un marco amarillo. Para modificar el tamaño o la posición del marco, arrastre (o deslice) este elemento de interfaz.
Puede indicar múltiples marcos.
Al crear una plantilla específica para su uso en la impresión de láminas, se crea un área de impresión (80 mm × 80 mm) y un área de edición (50 mm × 50 mm) en el centro de la plantilla.

 Plantillas específicas para la impresión en metal

ÁREA DE METAL
Crea un marco para insertar objetos (imágenes, imágenes prediseñadas y caracteres).
En este marco puede insertar cualquier cosa siempre y cuando el elemento introducido sea un objeto que se pueda añadir a cotodesign.
Pinche este botón para añadir un marco gris. Para modificar el tamaño o la posición del marco, arrastre (o deslice) este elemento de interfaz.

Puede indicar múltiples marcos.
Al crear una plantilla específica para su uso en la impresión de metal, se crea un área de impresión en metal (80 mm × 80 mm) y un área de edición (50 mm × 50 mm) en el centro de la plantilla.

 Plantillas específicas para el corte

ÁREA DE CORTE
Crea un marco para insertar imágenes prediseñadas.
Haga clic en este botón para añadir un marco rectangular rojo. Para modificar el tamaño o la posición del marco, arrastre (o deslice) este elemento de interfaz.
Puede indicar múltiples marcos.
