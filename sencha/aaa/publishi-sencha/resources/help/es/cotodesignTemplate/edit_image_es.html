IMAGEN

* Esto no se muestra en el caso de las plantillas específicas para el corte.

CATEGORÍA
Selecciona el tipo o género de imagen.
Para añadir imágenes personalizadas seleccionar «Original».

Lista de imágenes
Muestra una lista de las imágenes que se pueden usar.
Pinchar una imagen para introducirla en la pantalla de edición.



 Botones que se muestran cuando se selecciona «Original» en «CATEGORÍA».


Añade una imagen que se puede colocar en la plantilla.
Se muestra la plantilla para seleccionar cómo importar la imagen.


Usar este dispositivo
Añade una imagen guardada en el ordenador que se está utilizando.

Usar otro dispositivo
Añade una imagen guardada en una tableta, un smartphone u otro terminal de información

  1. Pinche en este botón para mostrar el primer código QR.
  2. Use un terminal de información para leer el código QR mostrado.
  3. Usar el SSID de lectura y la contraseña para conectar el terminal de información a la red.
    Si no puede leer el código QR, utilice el SSID y las cadenas de caracteres de contraseña que aparecen en la pantalla.
  4. Pinche en [SIGUIENTE] para mostrar el segundo código QR.
  5. Use un terminal de información para leer el código QR mostrado.
  6. Usar el terminal de información para acceder a la URL de lectura.
    Si no puede leer el código QR, acceda a la URL que se muestra en la pantalla.
  7. Cargar la imagen.
* Para las plantillas específicas para la impresión en láminas, las imágenes se muestran en dos colores (el color de la lámina y el blanco).
Para ajustar la parte coloreada y el umbral blanco, seleccione la casilla de verificación «EDITAR IMAGEN» y, a continuación, haga clic en la imagen.

* Para las plantillas específicas para la impresión en metal, las imágenes se muestran en escala de grises.
Para ajustar el brillo o extraer el contorno, coloque la imagen y, a continuación, haga clic en .
