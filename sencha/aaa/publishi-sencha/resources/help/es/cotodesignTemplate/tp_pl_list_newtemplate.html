Determinación del tamaño y del método de impresión de una plantilla

TIPO DE IMPRESIÓN
Seleccione el tipo de máquina con la que se imprimirá esta plantilla.
Cuando imprima en una impresora, seleccione «Impresión a color». Cuando utilice un decorador láser, seleccione «Imprimir lámina». Cuando utilice una impresora de metal, seleccione «Impresión en metal». Si utiliza una máquina de corte, seleccione «Corte».
* El «TIPO DE IMPRESIÓN» no puede modificarse al editar un template que se ha guardado.


 Cuando se selecciona «Imprimir lámina» para «TIPO DE IMPRESIÓN»

Habilitar contorno
Imprime los contornos de caracteres, imágenes e imágenes prediseñadas.
Al seleccionar esta casilla de verificación, los contornos se aclaran, pero el tiempo de impresión se alarga.

 Cuando se selecciona «Impresión en metal» para «TIPO DE IMPRESIÓN»

Material
Seleccione el material. El «Material» se ajusta en «Configuración del material» en cotodesign Tool.

Modo de impresión
Seleccione el modo para que coincida con el objetivo de impresión.
Las condiciones de impresión, como la velocidad y la fuerza de impresión, cambiarán dependiendo del modo seleccionado.
Las condiciones de impresión de cada modo se ajustan en «Configuración del material» en cotodesign Tool. 

Bidirección
Seleccione esta casilla de verificación para acortar el tiempo de impresión.

Habilitar contorno
Realiza una impresión metálica de los contornos de caracteres, imágenes y imágenes prediseñadas.

 Cuando se selecciona «Corte» para «TIPO DE IMPRESIÓN»

Material
Seleccione el material y el color del material. El «Material» se ajusta en «Configuración del material de corte» en cotodesign Tool.
