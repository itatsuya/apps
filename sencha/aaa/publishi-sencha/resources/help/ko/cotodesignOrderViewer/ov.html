주문 상태

1. 미결제 주문
결제가 완료되지 않은 주문 목록입니다.
대금을 받은 후에  을 클릭하십시오.
결제완료로 하면 "2. 인쇄 대기 중"란으로 바뀌며, cotodesign Print Manager에서 인쇄할 수 있게 됩니다.

  1. 대금을 받은 주문을 클릭합니다.
  2. 화면 아래쪽의 주문정보를 보고 고객의 주문인지를 확인합니다.
  3.  을 클릭합니다.
이 상태는 cotodesign Print Managerdml ［설정］창의 ［주문 폴더］탭에서 "지불확인 건너뛰기"가 OFF일 때 활성화됩니다.

2. 인쇄 대기 중
인쇄 대기 중인 주문 목록입니다.
cotodesign Print Manager를 사용하여 인쇄 처리로 넘어가면 "3. 인쇄 데이터 전송"란으로 바뀝니다.

3. 인쇄 데이터 전송
인쇄 중 또는 인쇄 완료된 주문 목록입니다.
고객에게 상품을 인도한 후  을 클릭하십시오.
납품완료로 하면 "4. 납품"란으로 바뀝니다.

  1. 상품을 인도한 주문을 클릭합니다.
  2. 화면 아래쪽의 주문정보를 보고 고객의 주문인지를 확인합니다.
  3.  을 클릭합니다.

4. 납품
고객에게 상품을 인도하고 모든 처리가 완료된 주문 목록입니다.
