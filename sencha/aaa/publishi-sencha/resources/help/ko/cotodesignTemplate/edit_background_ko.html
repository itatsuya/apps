배경

이미 상품에 들어가 있는 도안이나 문자, 혹은 상품 자체의 색을 지정하여 인쇄 전 상품의 상태를 화면에 재현합니다. 배경으로 삽입한 오브젝트는 화면에 표시될 뿐이며 인쇄는 되지 않습니다.
이렇게 함으로써 이미 있는 도안이나 문자를 피하여 인쇄하고 싶은 사진이나 문자를 배치하거나, 상품과 다른 색으로 함으로써 문자가 눈에 잘 띄게끔 할 수 있습니다.

cotodesign Client 활성화
cotodesign Client에서 고객이 인쇄 데이터를 커스터마이즈할 때에 배경 이미지 추가를 허가합니다.
허가하지 않을 경우에는 OFF로 합니다.

그리드 표시
Template의 왼쪽 아래를 기점으로 눈금(격자선)을 표시합니다.
가는 선은 5 mm 간격, 굵은 선은 25 mm 간격입니다.
"cotodesign Client 활성화"가 ON일 때 활성화됩니다.

［이미지］탭
배경으로 사용할 수 있는 이미지의 목록을 표시합니다.
클릭하면 편집 화면에 이미지가 삽입됩니다.

［색］탭
Template 전체를 지정한 색으로 착색합니다.

  • 색을 없애려면  을 클릭합니다.
  • "초기설정으로 복원": cotodesign Tool에서 설정된 "재질 색상"으로 되돌립니다.
    ※타각 전용 Template일 때만 표시됩니다.


Template에 배치 가능한 이미지를 추가합니다.
이미지 불러오기 방식을 선택하는 창이 나타납니다.

디바이스로부터 이미지 선택
조작 중인 컴퓨터에 저장되어 있는 이미지를 추가합니다.

디바이스로부터 이미지 업로드
태플릿이나 스마트폰 등 정보 디바이스에 저장되어 있는 이미지를 추가합니다.

  1. 클릭하면 1번째 QR코드가 표시됩니다.
  2. 표시된 QR코드를 정보기기로 읽습니다.
  3. 읽은 SSID와 패스워드를 사용하여 정보기기를 네트워크에 연결합니다.
    QR코드가 읽히지 않을 때는 화면에 표시된 SSID와 패스워드의 문자열을 사용하십시오.
  4. ［다음］을 클릭하면 2번째 QR코드가 표시됩니다.
  5. 표시된 QR코드를 정보기기로 읽습니다.
  6. 정보기기를 사용하여 읽은 URL에 접속합니다.
    QR코드가 읽히지 않을 때는 화면에 표시된 URL에 접속하십시오.
  7. 이미지를 업로드합니다.


이미지 목록을 새로 고침합니다.
