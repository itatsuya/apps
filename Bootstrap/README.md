# This is a skeleton of a web page using Node.js.

### usage
1. Clone this repository
```
$ git clone https://github.com/hikarut/NodeWebSkeleton.git
```
2. Installing packages
```
$ npm install
```
3. Build
```
$ npm run build
```
4. Start
```
$ npm run start
```
5. Open page
 * http://localhost:8080/
